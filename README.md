# gRPC Node.js To-Do App

A simple To-Do application built using gRPC and Node.js, following the Hussein Nasser crash course but updated for `@grpc/grpc-js`.

## Features
- Unary RPC for creating To-Dos
- Unary RPC for reading the To-Do list
- Protocol Buffers (`.proto`) for schema definition

## Setup
1. Clone the repo: `git clone <your-url>`
2. Install dependencies: `npm install`

## How to Run
1. Start the server: `node server.js`
2. In a separate terminal, run the client: `node client.js`
